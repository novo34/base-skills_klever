## Purpose
Construir interfaces escalables y mantenibles.

## Non-negotiables
- Uso de Server Components por defecto.
- Client Components solo para interactividad necesaria.

## Stop conditions
- "Use client" en la raíz de páginas o layouts de forma injustificada.

## Required Output
- Estructura de componentes optimizada.

## Verification
- Hidratación mínima en cliente.

## Purpose
Garantizar una estética visual superior (WOW effect) y accesibilidad.

## Non-negotiables
- Uso estricto de shadcn/ui y Tailwind CSS.
- Animaciones sutiles (framer-motion).

## Stop conditions
- Diseños básicos, sin gradientes o con tipografías por defecto.

## Required Output
- UI impecable con soporte Dark Mode.

## Verification
- El diseño se siente premium y dinámico.

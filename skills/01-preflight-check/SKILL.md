---
name: preflight-check
description: Fase de análisis mandatorio antes de cualquier implementación para detectar riesgos.
---

## Before implementation (Mandatory)
- Identificar información faltante (Missing Info).
- Identificar riesgos legales o de datos (Data Risks).
- Identificar posibles breaking changes.
- Proponer un plan de acción detallado antes de escribir código.
- Esperar confirmación del usuario si hay riesgos críticos o falta información esencial.

## Required Output
- Análisis de impacto (breve).
- Lista de posibles riesgos detectados.
- Plan de pasos detallado.
